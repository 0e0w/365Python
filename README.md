# 《365天深入理解Python》

## 0x01-项目说明 

这是一份自己准备用365天深入理解Python的学习进度清单，用于记录我在这一年时间里每天的学习收获。

本人编程零基础，只略懂一些Golang。因此本项目部分章节可能很基础，请见谅。

关于Python的其他资源，参考此项目：[https://github.com/0e0w/365Python](https://github.com/0e0w/365Python)
